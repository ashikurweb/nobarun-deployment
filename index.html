Deploy Nobarun to Live Server

A step-by-step guide to deploying the Nobarun project on a Digital Ocean Droplet. Follow these instructions carefully to ensure a stable production environment.

Steps
Server Prep Git & Code Dependencies Finalization

Phase 1: Server Configuration

Steps 1-2
1

Login and Select Domain

Go to DigitalOcean, log in to your account, and select the domain where you want to deploy the project.

Ensure you have SSH access ready for the terminal steps later.
2

Resize Droplet (Optimization)

When resizing a Droplet, to make the process faster and more stable, we perform the following actions:

  • Turn off the Droplet first.
  • Resize it to the desired specification.
  • Turn it back on.
Note: Another droplet size might be used temporarily for faster optimization during this process.

Phase 2: Pulling Code

Steps 3-5
3

Navigate to Project Directory

Open your terminal and SSH into the server, then navigate to the project root.

cd /home/ploi/bderp.online
4

Reset Local Changes

Discard any local changes to ensure a clean state for the new deployment.

git reset --hard
5

Pull Latest Code

Pull the latest changes from the repository. Assumes the default branch is main.

git pull origin main

Phase 3: Dependencies & Build

Steps 6-9
6

Update NPM Packages

If any NPM packages were added or updated in the recent commit, run one of the following:

npm install
npm update
7

Frontend Build

Compile the frontend assets for production. This prepares the static files.

npm run prod
8

Update Composer Packages

If any PHP backend packages were added or updated:

composer install
composer update
9

Run Database Migrations

If there are any new migration files, update the database structure using the tenant migrate command.

php artisan tenants:migrate

Phase 4: Cleanup & Finalize

Steps 10-11
10

Clear DOMPDF Cache

Delete the specific dompdf font family cache file from the storage directory to prevent rendering issues.

rm storage/fonts/dompdf_font_family_cache.php
11

Resize Droplet Back

Once the deployment is complete and tested, resize the Droplet back to its previous configuration to optimize costs and resources.

Deployment Complete
